1.✏️프로젝트 소개
------------------
### Work Mate 서비스
#### Todolist와 댓글을 통한 커뮤니케이션의 기능 결합으로, 합께 하루를 채워나가는 Todolist 프로젝트입니다.
</br>

### 기획의도

- Todolist를 통한 스케줄 관리
- 다양한 사용자들과의 의사소통

#### 기대효과
- 스케줄 관리
  - Todolist를 통한 손쉬운 스케줄 관리
- 의사소통
  - 다양한 사용자들과의 의사소통
- 동기부여
  - 사용자들과의 의사소통으로 동기부여
- 일상공유
  - 서로의 일상을 공유
</br></br></br>
